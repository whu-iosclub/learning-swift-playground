# 学习 Swift Playground

欢迎来到 Wuhan University iOSClub 的 Swift 教学资料仓库 —— learning-swift-playground。本仓库以 Playground 为载体，按主题组织示例与练习，适合自学与课堂教学。无论你是刚接触 Swift 的初学者，还是希望用 Playground 快速验证想法的进阶开发者，都能在这里找到清晰、动手可运行的示例。

---

### 项目简介

本仓库收集了武汉大学 iOSClub 团队整理的 Swift 教学内容，采用 Playground 的交互式形式，覆盖语言基础、常用 API、SwiftUI/布局、并发、网络与常见算法等主题。每个文件夹通常包含：概念讲解、示例代码、练习与解析（可运行）。

### 目标读者

- 大学课程与自学者：需要系统入门 Swift 的同学。
- iOS 开发初学者：想通过 Playground 快速熟悉语法与 API。
- 导师与助教：可直接用于课堂演示与布置作业。
- 希望用 Playground 做原型验证的开发者。

### 快速开始

1. 克隆仓库
   - 在终端中运行：
     git clone https://github.com/whu-iosclub/learning-swift-playground.git
2. 打开 Playground
   - macOS（Xcode）：
     - 直接在 Finder 中双击 .playground 或用 Xcode 打开。
   - iPad（Swift Playgrounds）：
     - 将相关 Playground 文件导入 Swift Playgrounds（支持的格式依 App 版本而定）。
3. 运行与编辑
   - Playground 会实时执行代码；建议逐步阅读注释、运行每个示例，修改并观察输出与 UI 变化。

### 推荐学习路径（从浅入深）

1. Basics — 熟悉变量、常量、类型、字符串与数值
2. ControlFlow & Collections — 掌握循环、条件、数组与字典
3. Functions & Closures — 理解函数、闭包与参数传递
4. Structs, Enums & Protocols — 学会设计数据模型与协议编程
5. Functional & Generics — 常用高阶函数与泛型编程
6. Concurrency — async/await、Task 和线程安全
7. SwiftUI — 声明式 UI 构建与布局
8. Networking & Testing — 网络请求、JSON 解析与单元测试
9. Projects — 综合练习与小项目实战

### 常见问题与提示

- Playground 卡顿或运行缓慢：关闭即时渲染或拆分成更小的 Playground 文件。
- 语法或 API 报错：请确认 Xcode/Swift 版本与仓库示例所用版本兼容。
- 想贡献或提出问题：请查看下方贡献指南与 Issue 模板。
