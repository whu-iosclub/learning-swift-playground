# SwiftUI 教学示例集

这是一个用于 Swift Playground 的 SwiftUI 教学项目，包含了丰富的示例代码，适合初学者学习 SwiftUI 和 Swift 编程。

## 📚 项目结构

### Swift 基础
- **Swift 语法入门** (`SwiftGrammarView.swift`)
  - 变量和常量
  - 基础数据类型
  - 集合类型（数组、字典、集合）
  - 控制流（if/for/while）
  - 函数定义和使用
  - 可选类型

### 基础组件
- **Hello World 示例** (`ContentView.swift`)
  - 最简单的 SwiftUI 视图
  - 基础组件使用

- **布局示例** (`LayoutExamplesView.swift`)
  - VStack - 垂直布局
  - HStack - 水平布局
  - ZStack - 层叠布局
  - 布局嵌套

- **颜色和渐变** (`ColorGradientView.swift`)
  - 基础颜色
  - 透明度
  - 线性渐变 (LinearGradient)
  - 径向渐变 (RadialGradient)
  - 角度渐变 (AngularGradient)
  - 自定义 RGB/HSB 颜色

### 交互和状态
- **状态管理** (`StateManagementView.swift`)
  - @State 属性包装器
  - @Binding 状态绑定
  - Toggle 开关
  - Slider 滑动条
  - 颜色选择器示例

- **计数器示例** (`StateManagementView.swift`)
  - 简单的状态管理
  - 按钮交互
  - 条件样式

- **计时器** (`timer.swift`)
  - Timer 使用
  - 开始/暂停/重置功能
  - 时间格式化
  - 按钮禁用状态

- **表单和输入** (`FormExampleView.swift`)
  - TextField - 文本输入
  - TextEditor - 多行文本
  - Picker - 选择器
  - DatePicker - 日期选择
  - 表单验证

### 列表和导航
- **列表示例** (`ListExampleView.swift`)
  - List 基础使用
  - Section 分组
  - ForEach 遍历

- **动态列表** (`ListExampleView.swift`)
  - 添加/删除项目
  - 拖动排序
  - 模态窗口 (sheet)
  - 待办事项示例

### 动画效果
- **基础动画** (`AnimationExamplesView.swift`)
  - 隐式动画 (.animation)
  - 显式动画 (withAnimation)
  - 弹簧动画
  - 旋转动画
  - 淡入淡出

- **手势和交互** (`AnimationExamplesView.swift`)
  - 点击手势 (TapGesture)
  - 拖动手势 (DragGesture)
  - 缩放手势 (MagnificationGesture)
  - 旋转手势 (RotationGesture)

## 🎯 学习路径建议

1. **Swift 基础** - 先了解 Swift 语法
2. **Hello World** - 理解基础视图结构
3. **布局示例** - 学习如何排列视图
4. **状态管理** - 掌握状态和交互
5. **计数器/计时器** - 实践状态管理
6. **列表示例** - 学习数据展示
7. **动画效果** - 增加交互体验
8. **表单示例** - 构建完整功能
9. **颜色和渐变** - 美化界面

## 💡 使用方法

1. 在 Swift Playgrounds 或 Xcode 中打开此项目
2. 运行项目，会看到主导航界面
3. 点击任意示例查看效果和代码
4. 每个示例都包含：
   - 实际运行效果
   - 代码注释
   - 学习要点说明

## 📖 教学要点

每个示例都包含：
- ✅ 清晰的代码注释
- ✅ 实用的功能演示
- ✅ 学习要点总结
- ✅ 最佳实践示例
